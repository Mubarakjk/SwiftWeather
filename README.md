A SwiftUI-based weather app that showcases a dynamic interface with day/night toggle and a 5-day forecast. I built this project while learning SwiftUI fundamentals from online tutorial
---

## Features

- Toggle between **Day and Night** modes  
- Display **current weather** with temperature and weather icon  
- Show a **5-day weather forecast**  
- Custom reusable components:
  - `WeatherDayView` for daily forecast
  - `MainWeatherStatusView` for current weather
  - `WeatherButton` for interactive UI  
- **Gradient background** transitions between day and night  
- Uses **SF Symbols** for weather icons
